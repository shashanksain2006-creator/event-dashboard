# Smart Event Dashboard

A modern and responsive Event Management Dashboard built using **HTML, CSS, and JavaScript**. The application allows users to create, manage, search, and delete events through a clean and interactive interface.

---

## Features

- Create new events
- Add event date
- Select event category
- Add event description
- Search events instantly
- Delete individual events
- Clear all events
- Live event counter
- Demo event generator
- Responsive design for desktop and mobile
- Smooth animations and modern user interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/smart-event-dashboard.git
```

### Navigate to the Project Folder

```bash
cd smart-event-dashboard
```

### Run the Project

Open the `index.html` file in your preferred web browser.

No additional installation or dependencies are required.

---

## How to Use

1. Enter the event title.
2. Select the event date.
3. Choose an event category.
4. Add a brief description.
5. Click **Create Event**.
6. View the event in the dashboard.
7. Use the search bar to filter events.
8. Delete individual events using the delete button.
9. Click **Clear** to remove all events.
10. Use the **Demo** button to generate a sample event.
